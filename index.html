AND.

AND is the AND Name Daemon, or another name daemon, or an Apache-licensed Named Daemon, or something else.

It's a progression from the thoughts I had when failing to write R Squared.

AND is written in C and uses hiredis to talk to its only supported backend, Redis.

Finally, the only targetted platform is Linux, and the build environment is on CentOS 7. If you want to help with some autoconf action, I'd appreciate it.
